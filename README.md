# Silex Bootstrapper

Silex application bootstrapper, just what you need to get in Silex world.


# Requirements

- Bower


# Install

Install Bower dependencies
```
$ bower install
```


# Versioning

Semantic versioning ([semver](http://semver.org/)) is applied to this project.
